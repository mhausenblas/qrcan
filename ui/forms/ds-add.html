Add data source ...
Name
Provide a name for the data source.
Access method
Define the access to the data source: you can either specify the location of an RDF document (in RDF/XML, Turtle, RDFa, etc.) or you provide a SPARQL Endpoint URI (see also the list of public SPARQL Endpoints and their status).
Access mode local remote
Specify the way the data source should be accessed: if you choose local, the data source will be downloaded to the local store. If you choose remote, the data source will be queried live.
